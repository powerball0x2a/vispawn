# Vispawn Examples

本目录包含 Vispawn 生成的示例可视化。

## 目录结构

```
examples/
├── pythagorean/     # 勾股定理示例
├── engine/          # 四冲程发动机示例
└── newton/          # 牛顿第二定律示例
```

## 使用方法

运行 Vispawn 服务后，访问 Web 界面即可生成这些示例。

## 贡献示例

欢迎提交你生成的优秀示例！请：

1. 创建以原理命名的文件夹
2. 添加 README 说明
3. 将生成的 HTML 文件放入文件夹
4. 提交 Pull Request
